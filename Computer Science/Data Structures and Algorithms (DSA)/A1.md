# Introduction
An algorithm must satisfy the following criteria:

**1. Input:** Zero or more quantities must be externally supplied.

**2. Output:** At least one quantity is produced.

***3. Definiteness:*** Each instruction is clear and unambigous. (also feasible.)

**4. Finiteness:** The algorithm must terminate after a finite number of steps.

**5. Effectiveness:** Should we simple enough, that in principle, it can be carried out by a person using a pen and paper.

The range of inputs for which the algorithm works must be specified carefully.
